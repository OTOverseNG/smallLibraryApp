Library App
===========

A simple JavaScript project to manage a personal library.  
You can add books with details (title, author, pages, and read status) and track whether youâ€™ve read them or not.  

This project is built to practice JavaScript object constructors and basic DOM manipulation.

Features
--------
- Add a new book (title, author, pages, read/not read)  
- Mark a book as Read or Not Read  
- Remove a book from the library  
- Simple UI to display the list of books  

Tech Used
---------
- HTML  
- CSS  
- JavaScript (with Object Constructors)
